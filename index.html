Bibit Reksadana API

Endpoint

https://bibit-reksadana.vercel.app/api/

Query Parameter

Parameter Type Default Description
search string "" Search query
page int 1 Page to retrieve
per_page int 25 Number of records per page
buy_from_bibit bool false Filter mutual funds that can be bought from Bibit
types string "" Filter mutual funds by types. Comma separated value from the following options:
  • equity
  • balanced
  • fixed_income
  • money_market
sharia bool false Filter mutual funds with sharia principle
usd bool false Filter mutual funds with USD currency
sort_by string "name" Sort the results by the given field. Possible values:
  • name
  • return_1d
  • return_1m
  • return_1y
  • return_3y
  • return_5y
  • return_ytd
  • drawdown_1y
  • aum
sort_direction string asc Sort direction
